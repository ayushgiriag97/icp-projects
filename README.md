# Ecomart-TeamProject

This project represents my first collaborative web development assignment built during ICP classes.  
The **Ecomart website** is a simple e-commerce platform showcasing teamwork in **HTML, CSS, and JavaScript**.  

## Features
- Product listing and categories  
- Basic navigation and styling with CSS  
- Team collaboration workflow
